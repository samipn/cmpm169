Experiment 2 - Vector Art, Animation & Interactivity

Imitate

I started with some code from the examples given to us on the generative p5 website. With that as my base I moved on to making my piece more chaotic.

Integrate

I used more interesting techniques that I found within multiple examples to make lines that come through in a random sort of way.

Innovate

I used the code to make something unique by allowing it to mesh together better. To accomplish this I added randomness wherever I could.
