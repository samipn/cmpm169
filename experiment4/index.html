Experiment 4 - Images, Video, & Sound Art

Imitate

Looking for inspiration, I was going for a pattern theme using video and pixels as the medium. I was able to find some examples that got the pixels the way I wanted, and so I used that as a base.

Integrate

Trying to fit the pieces together was tough, but I was able to mess around with the values of how the pixels were recorded and then I was able to make something that worked, but was very basic.

Innovate

To innovate, I took the code and made it work together, so it created an effect of both circular and square pixels, that when shown at different light levels changes the way it looks. It also has a pattern normally that gets effected by the light.
